\## Concepts Covered



\### Python List Slicing

List slicing allows you to cleanly reference sub-sections of sequential arrays using the notation `array\[start:stop:step]`.

\- \*\*First $n$ elements:\*\* `array\[:n]`

\- \*\*Last $n$ elements:\*\* `array\[-n:]`

\- \*\*Range bounds (inclusive/exclusive):\*\* `array\[start:stop]`
