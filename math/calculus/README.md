\## Mathematical Concepts Covered



\### Summation Notation ($\\Sigma$)

Summation notation is used to represent the addition of a sequence of numbers. A variable (the index of summation, e.g., $i$) updates sequentially across integers from a lower bound to an upper bound. 



Example evaluated in Task 0:

$$\\sum\_{i=2}^{5} i = 2 + 3 + 4 + 5$$
