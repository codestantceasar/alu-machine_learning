# Linear Algebra

This project introduces the fundamentals of linear algebra using Python and NumPy.

## Description

The project covers:

- Matrix and array slicing
- Matrix shapes
- Matrix transposition
- Element-wise operations
- Matrix addition
- Matrix concatenation
- Matrix multiplication
- NumPy operations

## Files

| File | Description |
|------|-------------|
| 0-slice_me_up.py | Slicing arrays |
| 1-trim_me_down.py | Extracting matrix columns |
| 2-size_me_please.py | Calculating matrix shape |
| 3-flip_me_over.py | Matrix transpose |
| 4-line_up.py | Adding arrays |
| 5-across_the_planes.py | Adding matrices |
| 6-howdy_partner.py | Concatenating arrays |
| 7-gettin_cozy.py | Concatenating matrices |

| 13-cats_got_your_tongue.py | NumPy concatenation |
| 14-saddle_up.py | NumPy matrix multiplication |
